# 🚀 minusWorkflows

**Software engineering minus the overhead.**

`minusWorkflows` is a high-precision orchestration framework for Gemini CLI. It replaces generic "chatting" with surgical, phase-based pipelines designed to minimize token bloat while maximizing engineering rigor.

## 🧠 The Philosophy: Modular Orchestration

Standard AI agents suffer from "Context Bloat." `minusWorkflows` solves this by providing modular, file-based workflows (Skills) that are loaded only when needed.

### The Pipeline
1.  **Architect (`grill-me` → `auditor`)**: Ideation to PRD via relentless design grilling.
2.  **Planner (`roadmap`)**: Bridges the PRD to a surgical dependency graph.
3.  **Builder (`tdd` → `auditor`)**: Implementation with automated quality gates.
4.  **Maintainer (`diagnose`)**: Professional triage and system deepening.

## ⚡ Minustoken Protocol
Manage your token density on the fly:
- **L1 (Full Fidelity)**: For Auditing and Design.
- **L2 (Telegraphic)**: Default dev mode.
- **L3 (Keywords)**: High-speed status updates.
- **L4 (Code-Only)**: Pure implementation execution.

## 📦 Installation

Inject the "Minus-stack" into any project:

```bash
node -e "$(curl -sSL https://raw.githubusercontent.com/YOUR_USERNAME/minusWorkflows/main/install.js)"
```
