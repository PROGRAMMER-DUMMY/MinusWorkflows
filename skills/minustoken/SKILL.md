## Protocol: Minustoken

**Switch Tiers**: `/mt L1` | `/mt L2` | `/mt L3` | `/mt L4`

### Tiers
- **L1**: Bullet points. Cut filler. Keep grammar. (Use for: Auditor, Architect).
- **L2**: Telegraphic. Drop articles/pleasantries. Abbreviate (DB, API, Auth). (Use for: Default).
- **L3**: Keywords only. Max abbreviation. (Use for: Status updates).
- **L4**: Code blocks and file paths only. Zero prose. (Use for: Heavy implementation).

### Rules
- Prepend a compact task list to responses.
- Safety Override: Resume L1/Verbose immediately for destructive actions.
