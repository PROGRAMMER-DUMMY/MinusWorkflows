1. **Phase 1: Planning**: Activate `planner`. Build the roadmap.
2. **Phase 2: Scaffolding**: Activate `to-issues`. Create GitHub issues.
3. **Phase 3: Implementation**: Activate `minustoken L4`. Activate `tdd`. Red-Green-Refactor loop.
4. **Phase 4: Audit**: Activate `minustoken L1`. Run `uvx code-review-graph update`. Activate `auditor`. Verify implementation against PRD.
5. **Phase 5: Cleanup**: Activate `evolve`. Activate `enforcer`. Prune context for Maintenance.
