1. **Phase 1: Planning**: Activate `planner`. Build the roadmap.
2. **Phase 2: Scaffolding**: Activate `to-issues`. Create GitHub issues.
3. **Phase 3: Implementation**: Activate `tdd`. Red-Green-Refactor loop.
4. **Phase 4: Audit**: Activate `auditor`. Verify implementation against PRD.
5. **Phase 5: Cleanup**: Activate `enforcer`. Prune context for Maintenance.
