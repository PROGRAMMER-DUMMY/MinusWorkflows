## Phase: Planning
Transform the high-level PRD into a surgical execution graph.

1. **Dependency Analysis**: Identify which components must be built first.
2. **Roadmap Scaffolding**: Create a sequence of "Tracer Bullets" (vertical slices).
3. **Context Requirements**: List exactly which files and skills will be needed for each step.

**Output**: A `.memory/ROADMAP.md` file containing the task graph and dependency list.
